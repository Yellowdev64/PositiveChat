# PositiveChat 💬🔒
A fully encrypted, serverless P2P chat app for Linux. Built with Python & PySide6.

## ✨ Features
- 🔐 End-to-end encryption (PyNaCl)
- 📱 QR code peer discovery for LAN connections
- 🎭 Positive placeholder messages (decrypt to reveal)
- 👤 Local profile system with setup wizard
- 🖥️ Modern dark-mode UI with message bubbles
- 📦 Zero dependencies on cloud servers

## 🐧 Installation
```bash
# 1. Clone the repo
git clone https://github.com/Yellowdev64/PositiveChat.git
cd PositiveChat

# 2. Make launcher executable
chmod +x run_positivechat.sh

# 3. Run (auto-creates virtual environment)
./run_positivechat.sh
